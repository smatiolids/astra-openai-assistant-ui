# Astra OpenAI Assistant UI

## Getting Started

Copy .env_sample to .env and set your environment variables

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
